# Customer Engagement Heatmap Analysis

This repository contains a Seaborn-based customer analytics visualization for executive insights.  
The heatmap shows correlations between key customer engagement metrics such as:

- Purchase frequency  
- Average order value  
- Days since last visit  
- Email click-through rate  
- Support usage  
- NPS score  

The visualization helps identify relationships between metrics to support customer experience strategy.
